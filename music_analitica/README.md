# Анализатор статистики прослушиваний

## Описание
Проект для анализа истории прослушиваний пользователя из файла JSON.

## Структура проекта
*   `music.json` - Файл с данными о треках.
*   `music_analytics.py` - Основной скрипт для анализа и визуализации.
*   `README.md` - Этот файл.

## Запуск проекта

1. Установите Python и библиотеку Matplotlib:
   `pip install matplotlib`
2. Запустите скрипт:
   `python music_analytics.py`
